PT Trainer App (Streamlit) — Package
====================================

실행 방법
---------
1) 가상환경 생성 후 활성화 (선택):
   python -m venv .venv
   # Windows
   .venv\Scripts\activate
   # macOS/Linux
   source .venv/bin/activate

2) 의존성 설치:
   pip install -r requirements.txt

3) 실행:
   streamlit run main.py

데이터 저장파일: workout_app.xlsx (앱 실행 경로에 자동 생성됩니다)
